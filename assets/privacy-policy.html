Privacy Policy

Top Love Hacks Publisher  ·  Last updated: May 3, 2026

1. Overview

Top Love Hacks Publisher ("the app", "we", "our") is a personal content publishing tool operated by a sole individual. It is used exclusively to schedule and publish original articles from the Top Love Hacks Medium publication (medium.com/@nanmaniliya53) to a Pinterest business account owned and operated by the same individual.

This application is not available to the public and does not interact with, collect data from, or provide any service to third-party users.

2. Data We Collect

This app does not collect, store, or process any personal data from any user other than the account owner. No data is shared with, sold to, or accessible by any third party.

The only data handled by this app is the account owner's own Pinterest credentials and OAuth access token, stored locally on a private machine and never transmitted to any external server beyond Pinterest's official API.

3. Pinterest API Usage

This app connects to the Pinterest API v5 solely to create and publish pins on the account owner's own Pinterest board. All API interactions comply with Pinterest's Developer Terms of Service. No data from Pinterest is stored, resold, or used for any purpose beyond publishing the account owner's content.

4. Cookies & Tracking

This app does not use cookies, tracking pixels, analytics, or any form of user tracking. There is no website, web interface, or front-end accessible to the public.

5. Third-Party Services

The app makes API calls to the following services on behalf of the account owner only:

  • Pinterest API v5 — to publish pins
  • Google Gemini API — to generate pin headline text from the owner's own article content

No personal data belonging to any third party is ever sent to these services.

6. Data Security

All credentials and tokens are stored as environment variables on the account owner's private machine or as encrypted secrets in a private code repository. No credentials are hard-coded or exposed publicly.

7. Children's Privacy

This app is not directed at children and does not knowingly collect information from anyone under the age of 13.

8. Changes to This Policy

This policy may be updated from time to time. The "Last updated" date at the top of this page will reflect any changes.
